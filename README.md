
# README

## Uvod

Ove mobilne aplikacije razvijene su u sklopu diplomskog rada i implementirane su u četiri klijentske tehnologije: Kotlin Jetpack Compose, Flutter, React Native i Ionic. Za potrebe rada korištene su dvije pozadinske platforme: Supabase i Firebase. Cilj izrade aplikacija je omogućiti usporedbu razvoja mobilnih aplikacija korištenjem različitih tehnologija.

---

## Instalacija aplikacije

U repozitoriju se u mapi `aplikacije` nalaze četiri APK-a pomoću kojih se aplikacije mogu instalirati na uređaj. Dostupna su četiri APK-a, svaki za jednu verziju aplikacije:

1. **Kotlin + Supabase**
2. **React Native + Firebase**
3. **Flutter + Firebase**
4. **Ionic + Supabase**

Za instalaciju APK-a potrebno je:

1. Preuzeti odgovarajući APK na Android uređaj.
2. Omogućiti instalaciju iz nepoznatih izvora u postavkama uređaja.
3. Otvoriti APK i pratiti upute za instalaciju.

Detaljnije upute za build i instalaciju APK-a za svaku pojedinu tehnologiju, kao i za postavljanje pripadajuće pozadinske platforme, nalaze se u zasebnim mapama unutar repozitorija.
